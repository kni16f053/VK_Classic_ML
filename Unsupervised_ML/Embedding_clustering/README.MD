# ДЗ №4 — Кластеризация эмбеддингов (без учителя)

## Что сделано
- Загружены эмбеддинги из `train.npz`
- Данные сжаты до 2D, 3D и 4D с помощью `TruncatedSVD`
- Проведена **нормализация** данных (проекция на сферу) для лучшего визуального анализа
- Выполнена кластеризация разными алгоритмами:
  - `KMeans` (2D, 3D, 4D)
  - `BisectingKMeans` (итеративный KMeans)
  - `SpectralClustering`
  - `AgglomerativeClustering`
  - `Birch`
  - `HDBSCAN`
  - `OPTICS`
- Результаты сохранены в `.csv` для загрузки на AllCups

## Лучший результат
- **Алгоритм:** `BisectingKMeans` с `n_clusters=5` на нормализованных 3D данных
- Почему: итеративное разбиение кластеров дало наиболее чёткое разделение точек на сфере

## Освоенные техники
- Понижение размерности (`TruncatedSVD`)
- Нормализация данных (проекция на единичную сферу)
- Визуализация 3D кластеров (`Plotly`)
- Подбор числа кластеров через визуальный анализ
- Сравнение 7 алгоритмов кластеризации

## Файлы
- `resultB3Dpp.csv` — лучший результат (BisectingKMeans, 3D, нормализация)
- Остальные `result*.csv` — эксперименты с другими алгоритмами

## Запуск
1. Положить `train.npz` в папку с ноутбуком
2. Запустить все ячейки
3. Загрузить нужный `.csv` на AllCups
