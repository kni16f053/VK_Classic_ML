# ДЗ №6 — Рекомендательные системы (кластеризация и совстречаемость)

## Что сделано
- Загружены и предобработаны данные о книгах и оценках пользователей (Book-Crossing dataset)
- Отфильтрованы выбросы: пользователи с <5 оценок, книги с <5 оценок, пользователи с >200 оценок
- Построена матрица оценок `(пользователь × книга)` в разреженном формате
- Реализованы два подхода к рекомендациям без учителя:

### 1. Кластеризация пользователей (пункт 2)
- Понижение размерности матрицы до 3D (`TruncatedSVD`)
- Нормализация данных (проекция на сферу)
- Кластеризация методом `BisectingKMeans` (n_clusters=5)
- Для каждого кластера вычислен топ-10 книг по средней оценке
- Выданы персональные рекомендации для пользователя vid=1000

### 2. Совстречаемость (пункт 3)
- Построены пары книг, встречающихся у одного пользователя
- Вес пары = полусумма оценок пользователя за обе книги
- Отобраны пары с суммарным весом > 16 (средняя оценка > 8)
- Реализована функция `getRecommendations(book_id)` для поиска связанных книг

## Освоенные техники
- Фильтрация выбросов в пользовательских данных
- Построение разреженных матриц (`coo_matrix`)
- Кодирование категориальных признаков (`LabelEncoder`)
- Понижение размерности (`TruncatedSVD`)
- Кластеризация (`BisectingKMeans`)
- Анализ совстречаемости товаров (market basket analysis)
- Визуализация кластеров в 3D (`Plotly`)
