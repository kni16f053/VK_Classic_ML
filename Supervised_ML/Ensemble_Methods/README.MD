# Домашнее задание №5: Прогнозирование цены квартир с использованием бустинга

## Обзор

Этот Jupyter Notebook представляет собой решение пятого домашнего задания, направленного на предсказание стоимости квартир на основе табличных данных. Основная цель — построение и сравнение различных регрессионных моделей, включая линейные модели, случайный лес и градиентный бустинг (XGBoost), для достижения наилучшей точности прогноза.

## Структура ноутбука

1. **Загрузка и первичный анализ данных:**
   - Импорт необходимых библиотек.
   - Загрузка тренировочного датасета (`train.csv`).
   - Анализ структуры данных, типов признаков и выявление пропусков.

2. **Предобработка данных:**
   - Удаление неинформативного столбца `id`.
   - Преобразование признака `date` в числовой хеш.
   - Заполнение пропусков в столбцах `metro_dist`, `g_lift`, `build_tech` средними значениями.
   - Анализ распределения целевой переменной `price` и её логарифмическое преобразование (`np.log1p`) для приближения к нормальному распределению.
   - Отбор и логарифмирование признака `area` для улучшения корреляции.

3. **Построение и оценка моделей:**
   - Разделение данных на тренировочную и тестовую выборки.
   - Обучение и оценка следующих моделей:
     - Гребневая регрессия (`Ridge`)
     - Линейная регрессия (`LinearRegression`)
     - Случайный лес (`RandomForestRegressor`)
     - Градиентный бустинг (`XGBoost`)
   - Сравнение моделей по метрике MAE (средняя абсолютная ошибка).

4. **Подготовка тестовых данных и формирование результата:**
   - Загрузка и аналогичная предобработка тестового датасета (`test.csv`).
   - Применение лучшей модели (XGBoost) для предсказания цен.
   - Формирование и сохранение выходного файла `result.csv` с предсказаниями.

## Используемые технологии

- `Python 3`
- `pandas` — для обработки и анализа данных.
- `numpy` — для численных расчетов.
- `matplotlib`, `seaborn` — для визуализации данных.
- `scikit-learn` — для предобработки данных и реализации моделей (Ridge, LinearRegression, RandomForestRegressor, метрики).
- `xgboost` — для реализации градиентного бустинга.

## Ключевые выводы

- **Лучшая модель:** Градиентный бустинг (XGBoost) показал наилучшие результаты на тестовой выборке с MAE ≈ 0.2654, что соответствует ошибке около 30% от логарифмированной цены.
- **Значимость предобработки:** Логарифмирование целевой переменной и признака `area` существенно улучшило качество моделей, приблизив распределения к нормальному.
- **Важность признаков:** Наибольшее влияние на цену квартиры оказали признаки `area` (площадь) и, в меньшей степени, количество комнат и этаж.

## Результаты

Итоговый файл с предсказаниями `result.csv` содержит два столбца:
- `id` — идентификатор квартиры из тестовой выборки.
- `price` — предсказанная цена квартиры.
