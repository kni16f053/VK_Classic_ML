# Домашнее задание №3: Классификация сайтов 18+ по текстовому описанию (Classic ML)

## Описание проекта

В этом проекте решается задача бинарной классификации веб-страниц по их заголовкам (`title`) и URL-адресам. Цель — определить, относится ли сайт к категории **18+**, используя методы классического машинного обучения.

Особенность задачи в том, что контент 18+ часто содержит специфические слова, написанные с ошибками или в произвольном порядке. Поэтому в работе сделан упор на **CountVectorizer** без учёта контекста, но с тщательной предобработкой текста.

---

## Основные этапы работы

### 1. Предобработка текста (`title`)

- Удаление стоп-слов (русские + английские)
- Токенизация по словам
- Эксперименты со **стеммингом** и **лемматизацией**:
  - Стемминг русских слов улучшил качество
  - Лемматизация не дала прироста, но замедлила работу
- Удаление слов длиной менее 2 символов

### 2. Предобработка URL

- Разделение URL на домены
- Удаление последнего домена (кроме явных маркеров 18+)
- Разбиение слов на **4-граммы** (и частичная замена на 3-граммы по заранее собранному словарю маркеров)
- Удаление шумовых символов

### 3. Формирование признаков

- Объединение обработанного URL и `title` в одну строку
- Векторизация через `CountVectorizer` (1-2 граммы)
- Классификатор: `SGDClassifier` (log loss, balanced class weight)

### 4. Оценка качества

- Метрика: **F1-score**
- Разбиение: train / validation = 50/50
- Итоговое качество на валидации: **~0.981**
